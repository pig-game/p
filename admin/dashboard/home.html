• Try to learn some coding

The coding languages used in Pig Game are actually quite easy to learn

Here are some of the coding languages used in Pig Game:

Python

Python is the language that the game itself(not the website) is coded in.

Python is the first coding language that I learned, and it still is probably the easiest of all of the coding languages that I know.

Python is a good coding language to get started with. It is very easy to learn and is not very complicated.

I found a little PDF about getting started with Python here.

HTML/CSS/JavaScript

These all go together to build websites.

They are slightly more difficult and I think Python would be a better choice for begining coding.

CSS(Cascading Style Sheets) is for adding the style to a page(positions, colours, fonts, e.c.t.).

HTML(Hyper-Text Markup Language) is what sets all the text, links and buttons.

Together, HTML and CSS can make a nice web page, and you virtually cannot make a web page without both.

You could make a very dull, boring one with just HTML but it would not be very eyecatching.

With just CSS, (as far as I'm aware) you could not make a website full stop.

JavaScript is what would handle logins, making sounds play, all that stuff that makes a web page come to life.

It is the most simmilar to Python (out of HTML, CSS and JavaScript) because it can do logic stuff (like 'do this if this is true').

It can take a fair while to learn all three of these, and I would highly recomend Python to learn first.

So, remember this. Python for making apps (and the game), HTML/CSS/JavaScript for website coding.
