Instructions for zip download

  1. In file explorer, right-click on the zip file
  2. Click on the option that says "Extract all"
  3. In the box that will pop up, type in the location of where you want the folder to be extracted to
  4. Click "extract"
  5. Go to the directory that you type in on step 3
  6. There should be a folder named the name of the zip file, open it
  7. In the foloder, open the application that is the name of the version
  8. If you followed the instuructions corectly, the version of Pig Game that you downloaded should open
