About the versions

Pig Game Demo Version

Pig Game Demo Version is a test to see if other versions Pig Game can run on your computer.

To control the pig in Pig Game Demo Version, use the < ^ > arrow keys (found at the bottom right of the keyboard on most keyboards)

Pig Game V1 PRO

Pig Game V1 PRO is still in development and will cost a price.

Pig Game V1 is a limited version of Pig Game V1 PRO but is free.
